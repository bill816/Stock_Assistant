Stock_Assistant
===============
##Stock Assistant是什么?
一个抓股票数据的Python脚本，使用Tencent的证券API。当前版本是0.1，缓慢开发中。


##Stock Assistant现在有哪些功能？

* 设置抓取间隔
* 从文本文件读取股票代码

##Dependencies

* Python 2.7.x
* colorama >= 0.2.5

##USAGE

命令行输入`python stock_assistant.py sh600577 sh600011 sh600001`  
或者在sample文件中写入股票代码，在图形界面下执行。
